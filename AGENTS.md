## 项目导航
- 简介：本项目是 Windows 本地离线中文改写工具，提供网页入口与 OpenAI 风格接口，核心依赖 `llama-server.exe` + GGUF 模型。
- 关键入口：`README.md` 说明安装、启动与排障；`启动.bat` 负责补齐模型、下载 llama.cpp、启动服务；`index.html` 为本地网页界面；`images/` 存放展示图。
- 常用命令：双击 `启动.bat` 启动整套流程；PowerShell 手动启动见 `README.md` 中 `llama-server.exe -m .\qwen3-merged-aigc_zhv3-Q4_K_M.gguf --host 127.0.0.1 --port 8181`；服务自检用 `Invoke-RestMethod http://127.0.0.1:8181/v1/models`。

## 项目约束
- 保持项目为“开箱即用”的 Windows 本地方案，改动启动链路时需兼顾普通用户双击启动场景。
- 涉及启动、下载、模型文件名、端口 `8181`、目录名时，必须同步核对 `README.md` 与 `启动.bat`，避免文档和脚本漂移。
- 网页入口默认由本地 `llama-server` 以 `--path .` 提供，若调整前端文件位置或访问路径，需同时校正启动参数与文档。
- 对外表述应保持“中文改写/润色/写作辅助”定位，不扩展为规避检测或合规审查用途。
